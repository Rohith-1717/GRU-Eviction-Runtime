
## Overview

This is a userspace virtual memory management system that implements and evaluates traditional and machine learning-based page replacement algorithms. Built using Linux's `userfaultfd` mechanism, it intercepts real page faults generated by memory accesses and resolves them entirely in userspace while using a trained GRU (Gated Recurrent Unit) neural network to predict future page reuse.


Unlike conventional algorithms that rely solely on recency or reference bits, the learned policy continuously captures workload behavior through a recurrent neural network and predicts which resident page is least likely to be accessed again.

---

## Features

- Userspace page fault handling using Linux `userfaultfd`
- Runtime GRU inference implemented entirely in C++
- Offline model training using JAX and Flax
- Multiple page replacement policies (LRU, Clock and Learned GRU)
- Synthetic workload generation (Sequential, Random and Zipfian)
- Swap-file-backed virtual memory
- Runtime collection of latency and swap statistics
- Modular architecture for experimenting with replacement algorithms

---

## Workload Generation

The project supports synthetic workload generation for evaluating page replacement algorithms.

Supported workloads:

- Sequential
- Random
- Zipfian

Each workload is stored as a trace file.

Offsets are relative to the beginning of the memory region allocated using `mmap()`.

## Performance Metrics

The project records:

- Total Page Faults
- Average Page Fault Latency
- Swap Read Latency
- Swap Write Latency
- Learned Eviction Count
- Cache Hit Ratio
- Cache Miss Ratio
