# CustomEvictor

## Overview

This is a userspace virtual memory management system that implements and evaluates traditional and machine learning-based page replacement algorithms. Built using Linux's `userfaultfd` mechanism, it intercepts real page faults generated by memory accesses and resolves them entirely in userspace while leveraging a trained GRU (Gated Recurrent Unit) neural network to predict future page reuse.

The project compares three page replacement policies:

- Least Recently Used (LRU)
- Clock (Second-Chance)
- GRU-Based Learned Page Replacement

Unlike conventional algorithms that rely solely on recency or reference bits, the learned policy continuously captures workload behavior through a recurrent neural network and predicts which resident page is least likely to be accessed again.

---

## Features

- Userspace page fault handling using Linux `userfaultfd`
- Runtime GRU inference implemented entirely in C++
- Offline model training using JAX and Flax
- Multiple page replacement policies (LRU, Clock and Learned GRU)
- Synthetic workload generation (Sequential, Random and Zipfian)
- Swap-file-backed virtual memory
- Runtime collection of latency and swap statistics
- Modular architecture for experimenting with replacement algorithms




## Swap Management

Modified pages are written to `swapfile.bin` using `pwrite()`.

Each evicted page is assigned a swap slot.

Whenever a swapped-out page is accessed again, it is restored using `pread()`.

---

## Workload Generation

The project supports synthetic workload generation for evaluating page replacement algorithms.

Supported workloads:

- Sequential
- Random
- Zipfian

Each workload is stored as a trace file.

Trace format:

```
<Operation> <Offset>
```

Example:

```
R 0
R 4096
W 8192
R 12288
```

Offsets are relative to the beginning of the memory region allocated using `mmap()`.

## Building

```bash
git clone <repository>

cd CustomEvictor

mkdir build
cd build

cmake ..

make
```

---

## Running

Train the GRU model

```bash
python trainer/train.py
```

Export the trained weights

```bash
python trainer/export_weights.py
```

Run the learned policy

```bash
./CustomEvictor --policy learned
```

Run LRU

```bash
./CustomEvictor --policy lru
```

Run Clock

```bash
./CustomEvictor --policy clock
```
